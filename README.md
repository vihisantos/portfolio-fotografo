## Sobre

**FrameStudio** e um template de portfolio premium desenhado para fotografos, videomakers e artistas visuais. Com foco em estetica (UI) e experiencia do usuario (UX), transforma visitantes em clientes de alto ticket. Layout dark mode nativo, espacamentos generosos, tipografia curada e animacoes suaves com Framer Motion.

---

## Tech Stack

### Frontend

| Tecnologia | Proposito |
|---|---|
| **React 19** | Biblioteca de interface |
| **TypeScript** | Tipagem estatica |
| **Vite 7** | Build tool e dev server |
| **Tailwind CSS 3** | Estilizacao utilitaria |
| **Framer Motion** | Animacoes fluidas (scroll reveal, transicoes) |
| **Lucide React** | Iconografia |
| **React Router 7** | Roteamento SPA |
| **next-themes** | Dark/Light mode |
| **Radix UI Slot** | Primitivo de composicao |
| **clsx + tailwind-merge + CVA** | Utilitarios de classe |

---

## Funcionalidades

- **Design Premium** — Dark/Light mode com next-themes, paleta de cores sofisticada
- **Galeria Interativa** — Masonry layout com lightbox em tela cheia e animacoes Framer Motion
- **Filtros Dinamicos** — Classificacao do portfolio (Casamento, Retrato, Editorial)
- **UX Cinematografico** — Scroll reveal com `whileInView`, glassmorphism na navegacao
- **Menu Mobile Imersivo** — Fullscreen com Backdrop Blur e entrada dinamica
- **100% Responsivo** — Otimizado do smartphone ao ultrawide 4K
- **Lead Generation** — Formulario de contato integrado a landing page
- **Performance Absurda** — Vite + React com carregamentos instantaneos
